AI School Project

This project involved creating an AI-powered racecar using the NEAT (NeuroEvolution of Augmenting Topologies) algorithm. The car was designed to autonomously navigate a racecourse, adjusting its behavior based on real-time sensor input, optimizing its driving strategy through evolutionary techniques. The goal was to demonstrate the power of AI in solving complex, dynamic challenges, such as autonomous driving.

Home Server Setup

The server serves as a test environment for various projects, including website hosting, file sharing, and other web services. I managed the server using Proxmox and Docker for containerization, ensuring high reliability, scalability, and security for all hosted applications. This project provided me with hands-on experience in server management, virtualization, and network configuration.
